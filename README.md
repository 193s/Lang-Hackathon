A JVM-based scripting language.
  
## Marg
An interpreter of Marg written in Java.
  
## Marg-Sharp
An implementation of Marg written in C#.  
Not supported.

# Language Documentation
  
###### Example
```
a = 0,
while (a < 10):
    a = a+1
;
 
```
  
  
### Comment
##### Single-line comment  
```
# comment
``` 
  
##### Multi-line comment  
```
#-- comment
comment*
--#
```

### Statement
Now you can use If and While statement like this:  
```
Reserved (Condition):
    Statements
;
```
